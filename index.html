Assignment

Assignment 1 - Basic HTML

This class covers the basics of HTML, focusing on the fundamental structure and elements that make up a web page. Students will learn how to create well-structured documents using tags, attributes, and formatting techniques. Additionally, the course will emphasize the importance of semantic HTML and best practices for ensuring accessibility and optimization.

Assignment 2 - Basic CSS

This class focuses specifically on CSS, exploring the principles of styling and layout for web development. Students will learn how to create visually appealing websites by mastering CSS features such as selectors, box models, flexbox, and grid layouts. The course will also emphasize the importance of responsive design and how to adapt styles for different devices while understanding client needs and preferences.

Assignment 3 - Page layout

This class focuses on page layout design, emphasizing the vital elements such as structure, spacing, and alignment. Students will learn how to create visually appealing layouts using CSS and HTML, understanding the principles of responsive design to ensure a seamless user experience across various devices. Additionally, the class will cover the importance of organizing content effectively and how to address client needs during the design process.
